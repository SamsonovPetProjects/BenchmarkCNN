#### Исследование классификации (CV Module)

**Цель:** Провести бенчмарк различных архитектур CNN на специфическом датасете EuroSAT для выбора оптимального энкодера (Encoder), способного эффективно распознавать типы поверхностей и ландшафт.

#### Архитектура исследования:

1. **Preprocessing Pipeline:** Реализована многоступенчатая обработка изображений (Resize 224x224, Normalization, ToTensor).
    
2. **Augmentation Strategy:** Для повышения обобщающей способности моделей применены случайные повороты (15°), отражения и цветовые искажения (ColorJitter).
    
3. **Class Mapping:** Оригинальные 10 классов EuroSAT агрегированы в 8 целевых групп (например, объединение различных типов посевов и пастбищ в единый класс "Crops") для стабилизации обучения.
    
4. **Transfer Learning:** Использован подход Fine-tuning. Веса моделей инициализированы из ImageNet, при этом базовые слои заморожены, а обучение ведется только для выходного линейного классификатора.
    
5. **Metrics Engine:** Система оценки включает в себя расчет Accuracy, построение Confusion Matrix (тепловых карт ошибок) и анализ ROC-AUC кривых для каждой архитектуры.

#### Текущие результаты бенчмарка:

- **EfficientNet-B0:** Показала лучшую семантическую логику, успешно разделяя природные и индустриальные зоны, несмотря на спектральную близость Grassland и Crops.
    
- **MobileNet-V3:** Продемонстрировала самую высокую скорость работы и точность в детекции линейных объектов (Highway), что критично для навигационных задач.
    
- **ResNet-50:** Обладает высокой мощностью, но склонна к нетипичным ошибкам при малом разрешении входных данных (64x64).
    
- **ResNet-18:** Определена как базовая модель (Baseline) с выраженным эффектом переобучения на малых выборках.
